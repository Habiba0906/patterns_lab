Structural Correlation

How independent processes can appear correlated due to shared trends, smoothing, and windowing.

Interactive Parameters

Memory Strength {{ "%.2f"|format(memory) }}
None Medium Strong

Controls autocorrelation. Higher values make each data point more dependent on previous values.

Trend Strength {{ "%.3f"|format(trend) }}
None Moderate Strong

Adds a linear trend to both series. Shared trends create apparent correlation.

Noise Level {{ "%.2f"|format(noise) }}
Low Medium High

Random noise added to each series. Higher values make the underlying patterns harder to see.

Smoothing Window {{ window }}
5 25 50

Number of points for moving average smoothing. Larger windows create smoother but potentially misleading trends.

Key Insight

These two time series are mathematically independent (generated separately), yet they appear correlated due to shared memory, common trends, and the smoothing effect. This demonstrates how correlation does not imply causation!

Spurious Correlation

Independent processes can appear correlated when they share common external influences or structural properties.

Shared Trends

Common trends, seasonal patterns, or smoothing can create apparent relationships between unrelated variables.

Smoothing Effects

Data smoothing and windowing operations can introduce artificial correlations between time series.
